Система контроля версий Git

17 марта 2026

Git — это распределённая система контроля версий, которую используют почти все разработчики. Она позволяет отслеживать изменения в коде, работать в команде и откатываться к предыдущим версиям.

Основные команды:

  • git clone — скопировать репозиторий
  • git add — добавить файлы в коммит
  • git commit -m "сообщение" — создать коммит
  • git push — отправить изменения на GitHub

В этой лабораторной работе мы активно используем Git для публикации сайта на GitHub Pages.
